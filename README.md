# Internship Risk Analysis App

## 📌 Project Description
This is a simple web application built using Python Flask.  
It checks the risk level of internship or company based on details like:
- Company Name
- Company Website
- HR Email
- LinkedIn presence
- Internship fee

The app classifies companies into:
- Low Risk
- Medium Risk
- High Risk

---

## 🚀 Features
- Simple web form input
- Risk analysis system
- Stores data in CSV file
- Easy to use interface
- Beginner-friendly Flask project

## 🛠️ Technologies Used
- Python
- Flask
- HTML
- CSS
- CSV file
